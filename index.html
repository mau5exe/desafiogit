Soda Stereo Web Site

Esta es una pagina creada por Mauro Torres, tributo a la mejor banda de Rock-Pop de latinoamerica: "Soda Stereo". Esta realizada con el unico fin de presentarla como proyecto final del curso "Desarrollo Web" de Coderhouse, y volcar todo lo aprendido desde el inicio al final del Modulo, usando herramientas como HTML5, CSS3, Bootstrap, SASS, Node JS, Git y GitHub mediante la tutoria de Roger Burbano, para el profesor Samuel Tocaimaza.
